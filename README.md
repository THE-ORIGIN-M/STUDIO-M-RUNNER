# STUDIO-M-RUNNER
En STUDIO M // RUNNER, asumes el papel de un agente infiltrado en la red informática más protegida del sistema. Tu objetivo es mantener la conexión el mayor tiempo posible mientras corres a través de una cuadrícula de datos infinitos y esquivas los potentes cortafuegos de seguridad (los bloques de neón rojos).

👾 Características Principales del Software:
Jugabilidad Arcade Pura: Mecánica de un solo toque (One-Touch). Solo necesitas calcular el momento exacto del salto para salvar los obstáculos.

Aumento de Frecuencia Progresivo: El juego integra un motor de dificultad algorítmica. Cada vez que tu puntuación sube 400 puntos, la velocidad del juego aumenta un 12% y la distancia entre cortafuegos disminuye, exigiendo reflejos sobrehumanos.

Optimización Multiplataforma: Diseñado mediante programación nativa en HTML5 Canvas, lo que permite un rendimiento ultra fluido a 60 FPS estables tanto en ordenadores (jugando con la barra espaciadora) como en dispositivos móviles y tablets (mediante pulsaciones táctiles en pantalla).

Estructura Escalable: El código base cuenta con buffers preparados para la futura integración de un sistema de base de datos para el High-Score global (ranking) y micro-pagos para la compra de vidas extras en caso de colisión.
